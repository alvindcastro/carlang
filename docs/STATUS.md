# Carlang Status

## Implemented

- Lexing and parsing
- Named combos
- Interpreter execution
- Spellbook slots `D` and `F`
- Core arithmetic and stack spells
- ASCII output through Sun Strike
- Bytecode compiler
- Bytecode VM
- CLI commands: `run`, `compile`, `vm`, `dis`, `repl`

## Partial

Block operations and conditionals are supported by the interpreter. The compiler currently focuses on stack and ASCII-output programs.
